# IPVault - Intellectual Property Registry

A blockchain-based platform for registering and licensing intellectual property rights, providing secure patent management and licensing marketplace.

## Features
- Register patents and IP rights as NFTs
- Secure licensing and royalty system
- Immutable IP ownership records
- Technology transfer marketplace

## Usage
Inventors use `register-patent` to secure IP rights, companies use `license-patent` to acquire usage rights with automated royalty payments.
